# React + Vite

Hello! I made this website as a way to learn Javascript XML as well as Tailwind CSS, while trying my best to create a sleek and modern design for my portfolio. I think this was great practice and I think it turned out amazing. Here's a list of some features my website contains:

- Navbar at the top: Features a navbar at the top of the screen with clickable buttons in which navigate to the selected button. It also has a frosted glass background which I thought was pretty nice.
- Custom cursor: Features a custom cursor that follows the real cursor position but in a smoothened way. It has a shadow underneath and when hovering over buttons such as the icon on the top left it expands and turns opaque.
- Animations: A huge portion of effort that went into this site is the animations using Framer Motion. I made sure that absolutely nothing in my website is static.
- 3d renders of my logo: There are 3d renders of prismatic glass throughout the website, which are also animated.
- Custom scrollbar: A custom scrollbar with a drop shadow that changes color when hovered over. I think this adds a lot of personality as personally I find the default scrollbar a little ugly.
